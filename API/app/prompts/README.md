# Mentorix Prompt Templates

This directory holds prompt template files loaded by `app.core.prompt_manager`.

## Naming Convention

Files are named `{feature}_{lang}.txt` where `lang` is a two-letter ISO code
(e.g. `en` for English, `hi` for Hindi).

## Files

| File | Purpose |
|------|---------|
| `content_generation_en.txt` | Subsection/chapter content generation |
| `test_generation_en.txt` | MCQ test generation |
| `explanation_en.txt` | Question explanation generation |
| `diagnostic_en.txt` | Diagnostic test MCQ generation |
